¿Qué es exactamente el curso Skill Code?

Es un entrenamiento intensivo de Desarrollo Web Front-End (La parte visual del internet). Está diseñado para principiantes absolutos, diseñadores gráficos que quieren digitalizar su trabajo, o emprendedores tecnológicos.

Aprenderás a usar editores de código profesionales (como Visual Studio Code) para escribir HTML5 (el esqueleto de la página) y CSS3 (la pintura, colores y diseño). Al finalizar, dejarás de ser un simple consumidor de internet para convertirte en un creador digital.

Alcances ¿Qué INCLUYE el temario?

  • Estructura HTML5: Dominio de etiquetas, textos, inserción de imágenes de alta calidad, botones y creación de enlaces de navegación entre páginas.
  • Estilos CSS3: Aplicación de colores corporativos, tipografías modernas (Google Fonts), márgenes, sombras y efectos visuales de transición.
  • Diseño Responsivo: Técnicas para que tu código detecte el tamaño de la pantalla y la página se acomode perfectamente tanto en celulares como en computadoras.
  • Despliegue Web: Te enseñaremos cómo subir tus archivos de código a un servidor gratuito para que tu página sea visible para cualquier persona en el mundo.

Lo que NO incluye este curso

  • Programación Lógica o Backend (Bases de datos, PHP, Python, JavaScript avanzado).
  • Creación de tiendas en línea con procesadores de pago reales. (Esto es parte de nuestros servicios de desarrollo Core Commerce).
  • Desarrollo de aplicaciones móviles nativas para iOS o Android.
