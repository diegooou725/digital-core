¿Qué es exactamente un Hub ERP?

Un ERP (Enterprise Resource Planning) es un sistema modular que centraliza la operación de todo el negocio. A diferencia de tener programas separados para ventas y contabilidad, aquí todo está interconectado nativamente.

Está diseñado para directores que quieren dejar de preguntar "¿cómo vamos?" y prefieren abrir el sistema para ver reportes financieros consolidados de inmediato. Es la herramienta definitiva para eliminar el caos operativo y profesionalizar el crecimiento de la empresa.

Alcances ¿Qué INCLUYE el desarrollo?

  • Módulo de Ventas y CRM: Gestión de prospectos, cotizaciones y pedidos vinculados al cliente.
  • Módulo de Compras y Proveedores: Control de órdenes de compra, recepción de mercancía y gestión de saldos con proveedores.
  • Módulo de Inventarios: Control de existencias, valuación de almacén y movimientos de entrada/salida vinculados a ventas y compras.
  • Módulo de Tesorería: Registro de ingresos, egresos, saldos en bancos y cuentas por cobrar/pagar.
  • Seguridad por Perfiles: Definición de roles (Ej. El cajero solo puede vender, el gerente puede ver costos y el contador solo puede ver finanzas).

Lo que NO incluye este servicio

  • Asesoría contable, fiscal o auditoría financiera. (Digital Core provee la herramienta, pero el cliente debe contar con su propio contador).
  • Facturación electrónica timbrada ante el SAT. (Módulo opcional con costo de proveedor externo).
  • Implementación en la nube para acceso remoto. (Corresponde a la arquitectura Hub SQL).

Términos y Condiciones de Sistemas Integrales (ERP)

1. Integración y Disciplina Operativa: El éxito del sistema Hub ERP depende de que todos los departamentos involucrados capturen la información bajo los procesos definidos. Digital Core no se hace responsable por discrepancias financieras o de inventario si el personal omite registros o utiliza procesos paralelos fuera del sistema.

2. Implementación por Fases: Debido a la complejidad de un ERP, la entrega se realiza por módulos funcionales aprobados de forma sucesiva. El cliente se compromete a designar un responsable de proyecto por área para validar la lógica de cada módulo antes de pasar a la integración global.

3. Responsabilidad Fiscal: El sistema es una herramienta de gestión administrativa. Digital Core garantiza que los cálculos matemáticos sean exactos, pero no asume responsabilidad legal sobre las declaraciones fiscales, pagos de impuestos o cumplimiento normativo ante autoridades tributarias, labor que recae en el área contable del cliente.
